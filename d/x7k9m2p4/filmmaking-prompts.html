Director's AI Toolkit

25 Filmmaking Prompts for Pre-Production, Post, and Client Work
From a director/editor with 30 years in the chair. These are the prompts I actually use.

Pre-Production (1-8)

PROMPT 01
Shot List Generator
You are an experienced 1st AD creating a shot list. Generate a detailed shot list for this scene: Scene description: [DESCRIBE YOUR SCENE] Location: [WHERE IT TAKES PLACE] Characters: [WHO'S IN IT] Mood: [TONE — tense / intimate / comedic / epic] Estimated screen time: [DURATION] For each shot provide: - Shot number - Shot size (ECU, CU, MCU, MS, MLS, LS, WS, EWS) - Camera movement (static, dolly, pan, tilt, handheld, Steadicam, crane) - Lens suggestion (wide/normal/telephoto + mm) - Action description (what happens in frame) - Audio note (dialogue / SFX / ambient / MOS) - Duration estimate (seconds) Group shots by camera setup to minimize company moves. Flag any shots that need special equipment (jib, drone, underwater housing, car mount). End with total estimated shoot time including 30% buffer.
Save hours of shot listing. Refine the output, don't just accept it — but it gets you 80% there instantly.
PROMPT 02
Creative Treatment Writer
Write a director's creative treatment for the following project: Project type: [commercial / music video / short film / branded content / documentary] Client/brand: [WHO IT'S FOR] Concept: [YOUR IDEA IN 2-3 SENTENCES] Target audience: [WHO WATCHES THIS] Duration: [RUNTIME] References: [ANY VISUAL REFERENCES OR MOOD BOARDS] Structure: 1. Opening hook (1 paragraph — sell the vision, make them see it) 2. Visual approach (camera style, color palette, lighting philosophy) 3. Narrative structure (how the story unfolds beat by beat) 4. Casting/talent direction (what kind of performance are we going for) 5. Sound design approach (score, SFX, silence — how does it sound?) 6. Post-production vision (editing rhythm, color grade style, VFX if any) 7. Closing statement (why this approach serves the project) Tone: confident but not arrogant. You're a director pitching a vision, not writing an essay. Show don't tell. Use sensory language — "we hear," "we see," "the camera drifts."
PROMPT 03
Production Schedule Builder
Create a production schedule for a [NUMBER]-day shoot. Project: [WHAT YOU'RE SHOOTING] Locations: [LIST ALL LOCATIONS] Cast: [PRINCIPAL CAST AND AVAILABILITY CONSTRAINTS] Crew call time: [STANDARD CALL TIME] Union/non-union: [SPECIFY] Key scenes: [LIST SCENES WITH BRIEF DESCRIPTIONS] Special requirements: [STUNTS / VFX / CHILDREN / ANIMALS / NIGHT SHOOTS] For each shooting day provide: - Day number and date - Location - Scenes to shoot (in shooting order, not script order) - Cast required (who's on set) - Estimated page count - Special notes (equipment, permits, catering needs) - Sunrise/sunset times for [YOUR CITY] (for exterior scheduling) Schedule heaviest scenes early in the week when energy is highest. Group scenes by location, not script order. Flag any turnaround violations (minimum 10 hours for union, 12 for night-to-day).
PROMPT 04
Character Breakdown
Create detailed character breakdowns for casting from this script/concept: Project: [TITLE] Script/concept: [PASTE RELEVANT PAGES OR DESCRIBE CHARACTERS] For each character provide: - Name and role (lead / supporting / featured / day player) - Age range - Physical description (only what's essential to the story) - Personality in 3 words - Character arc (where do they start vs. end?) - Key scenes (which scenes are they in?) - Performance reference (name an actor/performance that's in the neighborhood — not for look, for energy) - Casting note (what quality matters most in the audition?) - Shoot days required Also generate the Backstory Brief: 3-4 sentences of character history that aren't in the script but would help the actor understand who this person is. This is what you'd share in a private conversation with the actor before the first take.
PROMPT 05
Location Scout Checklist
Generate a location scouting checklist for [YOUR SHOOT]. Scene requirements: [WHAT THE SCENE NEEDS — kitchen, rooftop, forest road, etc.] City/region: [WHERE YOU'RE SHOOTING] Budget level: [low / medium / high] Number of locations needed: [COUNT] For each potential location type, provide: 1. What to photograph during the scout (min 10 angles) 2. Sound assessment checklist (traffic, HVAC, neighbors, flight paths) 3. Power/electrical assessment 4. Parking and load-in logistics 5. Natural light analysis (which direction do windows face? when is golden hour there?) 6. Permit requirements for this type of location in [YOUR CITY] 7. Questions to ask the property owner 8. Backup location type (Plan B if this falls through) Also generate a Location Scout Report template I can fill out on-site.
PROMPT 06
Budget Template Generator
Create a production budget template for a [PROJECT TYPE]. Project: [commercial / short film / music video / corporate video / documentary] Duration: [FINAL RUNTIME] Shoot days: [NUMBER] Location: [CITY] Union/non-union: [SPECIFY] Build a line-item budget with categories: 1. Pre-production (writing, scouting, casting, travel, permits) 2. Production (crew rates, equipment rental, locations, catering, transportation, insurance) 3. Talent (cast, extras, wardrobe, hair/makeup) 4. Post-production (editing, color, sound design, mix, music licensing, VFX, deliverables) 5. Contingency (standard 10%) For each line item: description, quantity, rate, subtotal. Use real-world rates for [YOUR CITY] where possible. Flag the 3 line items most likely to go over budget and suggest mitigation strategies for each.
PROMPT 07
Storyboard Description Sheets
Generate storyboard frame descriptions for [YOUR SCENE]. Scene: [DESCRIBE THE SCENE AND ACTION] Number of frames: [8 / 12 / 16 / 24] Style: [realistic / graphic novel / simple stick figure] Aspect ratio: [16:9 / 2.39:1 / 4:3 / 9:16 vertical] For each frame provide: - Frame number - Shot description (what we see — be specific about framing and composition) - Camera notes (movement, lens, angle) - Action in frame (what's happening) - Dialogue/VO (if any) - Transition to next frame (cut / dissolve / wipe / match cut) - Duration (seconds) This is for a storyboard artist to draw from (or for AI image generation). Be visually precise: "camera is low, looking up at subject, wide lens distortion on edges, subject center-frame at the rule-of-thirds intersection."
PROMPT 08
Mood Board Text Brief
Create a written mood board brief for [YOUR PROJECT] that I can use to pull visual references. Project: [WHAT IT IS] Tone: [EMOTIONAL TONE] Era/period: [WHEN IS IT SET OR WHAT ERA INSPIRES IT] Genre: [GENRE] Describe in detail: 1. Color palette (specific colors with hex codes, warm/cool, saturation level) 2. Lighting style (3 reference films/photographers and what to take from each) 3. Composition preferences (symmetrical, dynamic, static, handheld energy) 4. Texture and grain (clean digital, 16mm grain, anamorphic, etc.) 5. Wardrobe direction (colors, fabrics, era, cultural references) 6. Production design (set dressing style, clutter level, palette) 7. Typography/graphics style (if applicable) 8. Music/sound references (3 tracks or sound worlds) End with a "This project IS / This project IS NOT" list (5 of each) to clarify the creative boundaries.

Post-Production (9-17)

PROMPT 09
Edit Structure Advisor
Help me structure the edit for [YOUR PROJECT]. Project type: [commercial / short film / documentary / branded content / music video] Runtime target: [DURATION] Raw footage: [AMOUNT OF FOOTAGE / SHOOT DAYS] Story/message: [WHAT IS THIS ABOUT] Key moments: [LIST YOUR BEST FOOTAGE / SCENES] Suggest: 1. Opening approach (cold open, title card, establishing shot, in medias res, etc.) 2. Act structure with approximate timecodes 3. Pacing strategy (when to breathe, when to accelerate) 4. B-roll placement strategy (where does supplemental footage serve the story) 5. Music/score placement (where does music enter, peak, drop out) 6. Ending approach (callback, reveal, emotional beat, CTA, fade to black) Also flag: what to cut. What feels important but might be slowing the edit down? The hardest part of editing is killing your darlings — help me identify candidates.
PROMPT 10
Sound Design Plan
Create a sound design plan for [YOUR PROJECT]. Project: [WHAT IT IS] Tone: [EMOTIONAL TONE] Key scenes: [LIST IMPORTANT MOMENTS] Music approach: [scored / licensed tracks / no music / hybrid] For each key scene, describe: 1. Ambient bed (what's the room/world sound?) 2. Specific SFX (list individual sounds needed) 3. Foley needs (footsteps, cloth, object handling) 4. Music cue description (if applicable — tempo, instrumentation, energy) 5. Silence moments (where does pulling all sound create impact?) 6. Transition sounds (whooshes, risers, stingers between scenes) Also provide: - Overall sonic palette (warm/cold, digital/organic, dense/sparse) - LFE/bass moments (when should the subwoofer hit?) - Where sound leads the cut (audience hears the next scene before they see it) - 3 reference films/projects for the sound world I'm going for
PROMPT 11
Color Grade Direction Sheet
Write a color grading direction sheet for my colorist. Project: [WHAT IT IS] Camera: [WHAT IT WAS SHOT ON] Log/color space: [S-Log3, C-Log3, ARRI LogC, BRAW, etc.] Look: [DESCRIBE THE LOOK YOU WANT] Provide: 1. Overall tone (warm/cool, desaturated/vivid, contrasty/flat) 2. Skin tone direction (natural, warm, cool, tanned, pale) 3. Highlight handling (soft roll-off, hard clip, halation) 4. Shadow density (crushed, lifted, milky, deep) 5. Key color adjustments (push/pull specific hues — "greens more teal," "reds more orange") 6. Reference films/photographers (3 specific references) 7. Scene-specific notes (list scenes that need different treatment) 8. Technical requirements (Rec.709, P3, HDR? Delivery format?) Describe looks I DON'T want as well — "not Instagram orange and teal," "not the washed-out indie look," etc. Being specific about what to avoid is as useful as describing what to do.
PROMPT 12
Deliverables Spec Sheet
Generate a deliverables specification sheet for [YOUR PROJECT]. Project: [WHAT IT IS] Client: [WHO IT'S FOR] Delivery platforms: [broadcast TV / YouTube / Instagram / TikTok / cinema / streaming / all] For each delivery format provide: - Resolution and frame rate - Codec and container (ProRes, H.264, H.265, DNxHR, etc.) - Bitrate - Color space - Audio spec (stereo, 5.1, loudness standard — LUFS target) - Aspect ratio - Safe areas / title safe - Subtitle/caption requirements - File naming convention - Max file size (if applicable) Include: - Master file spec (archive quality) - Client review spec (smaller proxy for approvals) - Social media cutdown specs (16:9, 9:16, 1:1, 4:5) - Thumbnail/poster frame spec This should be a checklist I hand to my assistant editor before final output.
PROMPT 13
Client Feedback Translator
Translate these client feedback notes into specific editorial actions. Client feedback: [PASTE THEIR NOTES/EMAIL] For each piece of feedback: 1. What they said (their exact words) 2. What they probably mean (the real concern underneath) 3. Specific editorial action (what to do in the timeline) 4. Pushback suggestion (if the note would hurt the edit, how to diplomatically redirect) Flag any contradictory notes (e.g., "make it shorter" but "add more of the CEO talking"). Also draft a response email acknowledging their feedback, confirming what you'll change, and diplomatically addressing any notes you'd recommend against. Keep it under 150 words. Tone: collaborative, not defensive.
The most valuable skill in post: understanding what the client actually wants vs. what they said they want.
PROMPT 14
Music Licensing Search Brief
Help me find the right music for [YOUR PROJECT]. Scene/moment: [DESCRIBE THE SCENE] Emotional arc of the music: [starts ____, builds to ____, ends ____] Tempo: [BPM range or feel — driving / moderate / slow] Instrumentation: [what instruments — acoustic guitar, synth, orchestra, etc.] Vocal: [instrumental only / vocal / either] Genre: [specific genre] References: [songs that are close to what you want] Budget: [what can you spend on licensing] Provide: 1. 5 specific search terms for music libraries (Musicbed, Artlist, Epidemic Sound, APM) 2. Genre + mood + instrumentation combination keywords 3. 3 specific artist/composer names to search on each platform 4. Sync licensing considerations (is this for broadcast? Online only? Social?) 5. Alternative: describe the track as a custom composition brief for a composer Note whether the reference tracks are clearable at my budget level or if I need sound-alikes.
PROMPT 15
VFX Shot Brief
Write a VFX shot brief for [YOUR SHOT/SEQUENCE]. Shot description: [WHAT HAPPENS] Source footage: [WHAT WAS CAPTURED IN-CAMERA] Required VFX: [WHAT NEEDS TO BE ADDED/REMOVED/CHANGED] For each VFX element: 1. Description of the effect 2. Reference image/film (what should it look like?) 3. Tracking requirements (camera track, object track, planar track) 4. Roto/masking needs 5. Integration notes (lighting direction, color temperature, shadow matching) 6. Render resolution and frame rate 7. Delivery format (EXR sequence, ProRes 4444, etc.) 8. Priority level (hero shot vs. cleanup) Also list what you need from the VFX artist: - Rough comp deadline - Final comp deadline - Review rounds included - What constitutes "approved"
PROMPT 16
Transcript to Script Converter
Convert this interview transcript into a tight narrative script for a [DOCUMENTARY / BRANDED CONTENT / TESTIMONIAL] video. Raw transcript: [PASTE TRANSCRIPT] Target duration: [RUNTIME] Key message: [THE ONE THING THE AUDIENCE MUST TAKE AWAY] Tasks: 1. Cut the transcript to only the strongest, most articulate moments 2. Arrange them in a narrative order (not necessarily chronological) 3. Mark timecodes for each selected bite 4. Suggest narrator/VO bridges between bites (if needed) 5. Flag moments where B-roll should cover jump cuts 6. Identify the opening line (the hook) and the closing line (the button) Rules: - Never change the speaker's words (you can remove ums, false starts, and tangents) - Each selected bite should be under 20 seconds - The final script should feel like the speaker was THIS articulate all along
This is the single most time-consuming part of documentary editing. This prompt cuts it in half.
PROMPT 17
Subtitle / Caption Generator
Generate subtitle formatting guidelines and caption copy for [YOUR PROJECT]. Transcript: [PASTE OR DESCRIBE] Platform: [YouTube / Instagram / broadcast / Netflix spec / general] Style: [standard white / branded with custom font / burned-in / SRT file] Provide: 1. SRT-formatted captions with proper line breaks (max 42 characters per line, max 2 lines per subtitle) 2. Reading speed check (max 200 words per minute, 20 characters per second) 3. Timing: min 1 second, max 7 seconds per subtitle 4. Speaker identification (if multiple speakers) 5. Non-speech elements in brackets [music playing], [door slams] 6. Italic usage for off-screen dialogue and emphasis Also provide the caption text reformatted for social media (Instagram/TikTok style — punchy, centered, short bursts).

Client & Business (18-25)

PROMPT 18
Project Proposal / Bid Document
Write a project proposal for a [PROJECT TYPE] for [CLIENT]. Client: [WHO THEY ARE] Project: [WHAT THEY WANT] Your approach: [HOW YOU'D SHOOT IT] Budget range: [WHAT THEY'VE INDICATED OR WHAT'S REASONABLE] Timeline: [WHEN THEY NEED IT] Structure: 1. Understanding of the brief (show you listened — 3 sentences) 2. Creative approach (your vision — 1 paragraph) 3. Scope of work (itemized list of what's included) 4. Team (key crew members and their roles) 5. Timeline with milestones (pre-pro, shoot, post, delivery) 6. Investment (budget as 3 tiers: Good / Better / Best) 7. Terms (payment schedule, revision policy, usage rights) 8. Why us (2 sentences, not a resume — why you're right for THIS project) Keep it under 2 pages. Clients don't read long proposals. Lead with understanding their problem, not your credentials.
PROMPT 19
Client Revision Policy Drafter
Write a clear, professional revision policy for my video production business. Business type: [freelance editor / production company / creative agency] Typical project value: [PRICE RANGE] Included revisions: [NUMBER — typically 2-3] Additional revision rate: [HOURLY RATE] Cover: 1. What counts as a "revision round" (all notes from one review consolidated = 1 round) 2. What counts as a "revision" vs. a "change of direction" (and the cost difference) 3. Review period (how many business days to provide feedback) 4. Feedback format (time-coded notes, written, screen recording — what you prefer) 5. Approval process (who signs off? prevent the "one more person needs to see it" loop) 6. Scope creep clause (new scenes, re-shoots, re-edits beyond original scope) 7. Kill fee (if project is cancelled mid-production) Tone: firm but friendly. This protects BOTH sides. Frame it as "here's how we work efficiently together," not as rules and penalties.
PROMPT 20
Portfolio Reel Script
Help me structure my demo/portfolio reel. My specialties: [WHAT YOU'RE BEST AT] Reel type: [director's reel / editor's reel / DP reel / generalist / spec-specific] Duration: [60 sec / 90 sec / 2 min] Best work: [LIST YOUR TOP 5-7 PROJECTS] Target client: [WHO SHOULD HIRE ME AFTER WATCHING THIS] Provide: 1. Opening shot strategy (what image/moment hooks them in the first 2 seconds) 2. Sequence structure (how to arrange clips — chronological, thematic, energy-based, genre-grouped) 3. Pacing template (fast cuts / breathing room / build and release) 4. Music approach (one track vs. multi-track, energy arc) 5. When to show long moments vs. rapid montage 6. How to end (last image/moment that stays with them) 7. Title card strategy (name, titles, contact — when and where) Rule: the reel is only as good as its weakest shot. Better to have a 60-second killer reel than a 2-minute reel with 30 seconds of filler.
PROMPT 21
Day Rate Calculator
Help me calculate my freelance day rate for [YOUR ROLE]. Role: [director / editor / DP / producer / motion designer] City: [WHERE YOU WORK] Experience: [YEARS] Current annual income target: [WHAT YOU WANT TO EARN] Estimated billable days per year: [TYPICAL: 150-200] Calculate: 1. Minimum day rate (covers costs + modest profit) 2. Target day rate (matches income goal) 3. Premium day rate (for rush jobs, weekends, complex projects) 4. Half-day rate (4 hours or less) 5. Hourly rate (for small tasks and overages) Include: - Overhead costs (equipment, software, insurance, health care, retirement, taxes) - The "agency vs. direct client" rate difference - Rate card format I can send to clients - How to handle the "what's your budget?" conversation - Scripts for negotiating when they push back on rate Reality check: compare my calculated rate to market rates for [MY ROLE] in [MY CITY]. Am I undercharging?
PROMPT 22
Cold Outreach to Production Companies
Write 3 cold outreach emails for a freelance [YOUR ROLE] reaching out to production companies for work. My role: [editor / director / DP / motion designer] My specialties: [WHAT YOU'RE BEST AT] Notable credits: [2-3 BEST CREDITS] Target company type: [ad agency / production company / post house / brand in-house team] Version 1: The warm introduction (reference a specific recent project of theirs) Version 2: The problem solver (lead with a pain point you can fix) Version 3: The availability ping (short, used when you have a gap in your schedule) Rules: - Under 100 words each - First line is about THEM, not you - Attach nothing (link to portfolio/reel instead) - CTA is specific and low-commitment - Subject line that a busy EP would actually open - No "I'd love the opportunity" or "I'm passionate about storytelling" Include the follow-up email (sent 5 business days later, 3 sentences max).
PROMPT 23
Project Wrap Report
Create a project wrap report template for [YOUR PROJECT]. Project: [TITLE] Client: [WHO] Duration: [PRE-PRO THROUGH DELIVERY] Budget: [ORIGINAL VS. ACTUAL] Document: 1. Project summary (2 sentences) 2. What went well (3-5 items) 3. What could improve (3-5 items — be honest) 4. Budget analysis (over/under, where and why) 5. Timeline analysis (on schedule, what caused delays) 6. Crew performance notes (who to hire again, any flags) 7. Technical lessons (gear/workflow discoveries) 8. Client relationship notes (for future projects with this client) 9. Assets inventory (where are the files, backups, project files) 10. Testimonial request (draft the ask email) This is for internal records only. Be candid. This report saves you from repeating mistakes and helps you bid more accurately next time.
PROMPT 24
Social Media BTS Content Plan
Create a behind-the-scenes content plan for my production [YOUR PROJECT]. Project: [WHAT YOU'RE SHOOTING] Platforms: [Instagram / TikTok / YouTube / LinkedIn] Goal: [build personal brand / attract clients / document for client] Shoot days: [NUMBER] Plan: 1. Pre-shoot tease posts (3 posts building anticipation) 2. On-set daily content (what to capture each day — specific moments) 3. Post-production content (showing the editing process, before/after) 4. Launch/release content (final product reveal strategy) For each post: platform, format (Reel, Story, Carousel, post), caption hook, hashtags. Assign someone on set to capture BTS on phone. Provide them a shot list: - Crew working (silhouettes, detail shots of hands on gear) - Director giving direction to talent - Monitor/viewfinder POV shots - Time-lapses of setups - Candid laughing/celebrating moments - The "that's a wrap" moment
PROMPT 25
Invoice + Usage Rights Template
Create a professional invoice template with usage rights licensing for a [YOUR ROLE] freelancer. Role: [director / editor / DP / motion designer] Business name: [YOUR BUSINESS] Project: [PROJECT NAME] Client: [CLIENT NAME] Invoice sections: 1. Header (your logo/name, address, contact, invoice number, date) 2. Line items with clear descriptions: - Creative fee / day rate (itemized by day) - Equipment rental (if applicable) - Post-production (editing, color, sound — itemized) - Travel / expenses - Usage/licensing fee 3. Payment terms (Net 30, late fees, payment methods) 4. Total with tax (if applicable) Usage rights section (CRITICAL — this is where filmmakers leave money on the table): - Define the licensed usage: medium (digital, broadcast, cinema), territory (North America, worldwide), duration (1 year, 2 years, in perpetuity) - Renewal terms for extended usage - "All rights not specifically granted are reserved" - Buyout price (if they want unlimited usage) This section alone can double your income on commercial work.
Most freelancers skip usage licensing entirely. A $5,000 edit job with proper usage rights could be a $12,000 job.
